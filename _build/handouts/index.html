Exploring New Visualization Options

Introduction

This exercise will require that you choose 3 new visualization skills from the Pluralsight course Pandas Playbook: Visualization to explore further.

Setup

First, download the materials for this exercise on Frodo. Included in the download are all the datasets utilized in the Pluralsight course referenced above.

Then, set up your Jupyter notebook development environment the same way you have been using throughout this unit.

Building Habits

You may have noticed that throughout this unit, instructions for setup have been less and less detailed. This is intentionalâ€“ as you get more practice with the technologies required for Data Analysis, Interpretation, and Visualization, you should also aim to be able to quickly set up a new project without referring to documentation.

Today, try to set up your environment without referring to previous exercises. Simply the act of trying to remember will help ensure that these setup skills are accommodated into your long-term memory. This will be critical to you in your interview process and future career as a data analyst.

Exploration

In this Pluralsight course, you saw some new visualization techniques that could be used in various ways to show, explore, or analyze data.

  1. Choose your favorite dataset from the Pandas Playbook: Visualization course to work with in this exercise. You have all datasets available to you in the Frodo materials you downloaded in the previous section.

  2. Produce a set of at least 5 visualizations for the dataset you chose, using at least 2 of the following new technologies in your visualizations.

Technology List

  • Seaborn

  • bokeh

  • The Matplotlib OO API

  • pyplot

  • Customized styling with pandas

When you are finished, make sure to add your Jupyter notebook to a Github repo and push it to turn in this exercise.
